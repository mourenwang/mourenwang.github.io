基础数据类型之List

2020-03-04
Python3

列表

列表（list）是Python中最有用的数据类型之一。与字符串类似，list是由多个值组成的序列。在字符串中，每个值都是字符；而在列表中，值可以是任何数据类型。列表中的值称为元素（element）或项（item） 。

创建新列表的方法有多种；最简单的方法是用方括号( [ 和 ] )将元素包括起来:

[1, 2, 3, 4]
['sing', 'jump', 'rap','basketball']

第一个例子是包含4个整数的列表。第二个是一个包含4个字符串的列表。 当然，一个列表中的元素可以不需要是相同的数据类型。例如：

['spam', 2.0, 5, [10, 20]]

一个列表在另一个列表中，称为嵌套（nested）列表。
一个不包含元素的列表被称为空列表；你可以用空的方括号 [] 创建一个空列表。
正如你想的那样，你可以将列表的值赋给变量：

cxk = ['sing', 'jump', 'rap','basketball']
numbers = [42, 123]
empty = []
print(cheeses, numbers, empty)

其结果为：

[‘Cheddar’, ‘Edam’, ‘Gouda’] [42, 123] []

列表是可变的

访问列表中元素的语法，与访问字符串中字符的语法相同，都是通过方括号运算符实现的。 括号中的表达式指定了元素的索引。记住，索引从0开始：

print(cheeses[0])

其结果为：

‘Cheddar’

和字符串不同的是，列表是可变的。当括号运算符出现在赋值语句的左边时，它就指向了列表中将被赋值的元素。

numbers = [42, 123]
numbers[1] = 5
print(numbers)

其结果为：

[42, 5]

numbers中索引为1的元素，原来是123，现在变成了5。任何整数表达式都可以用作下标。如果你试图读或写一个不存在的元素，你将会得到一个索引错误(IndexError)。如果下标是负数，它将从列表的末端开始访问列表。

in运算符在列表中同样可以使用。

cxk = ['sing', 'jump', 'rap','basketball']
print('basketball' in cxk)
print('pingpong' in cxk)

其结果为：

True
False

遍历列表

最常用的遍历列表的方式是使用for循环。语法和字符串遍历类似：

cxk = ['sing', 'jump', 'rap','basketball']
for i in cxk:
    print(i)

如果你只需要读取列表中的元素，这种方法已经足够。然而，如果你想要写入或者更新列表中的元素，你需要通过下标访问。一种常用的方法是结合内置函数 range和 len ：

for i in range(len(cxk)):
    cxk[i] = cxk[i] * 2         #双倍的快乐【/手动狗头】

这个循环将遍历列表并更新每个元素。len 返回列表中的元素个数。range 返回一个包含从0到n−1下标的列表，其中n是列表的长度。 每次循环中，i 得到下一个元素的下标。循环主体中的赋值语句使用 i 读取该元素的旧值，并赋予其一个新值。

对一个空列表执行for循环时，将不会执行循环的主体：

for x in []:
    print('This never happens.')

尽管一个列表可以包含另一个列表，嵌套的列表本身还是被看作一个单个元素。 下面这个列表的长度是4：

['spam', 1, ['Brie', 'Roquefort', 'Pol le Veq'], [1, 2, 3]]

列表操作

运算符拼接多个列表：

a = [1, 2, 3]
b = [4, 5, 6]
c = a + b
print(c)

其结果为：

[1, 2, 3, 4, 5, 6]

运算符 * 以给定次数的重复一个列表:

print([0]*3)
print([1, 2, 3] * 2)

其结果为：

[0,0,0]
[1, 2, 3, 1, 2, 3, 1, 2, 3]

列表切片

切片（slice）运算符同样适用于对列表：

a=[1,2,3,4,5,6]
print(a[1:3])
print(a[:4])
print(a[3:])
print(a[::-1])
print(a[:])

其结果为：

[2,3]
[1,2,3]
[4,5,6]
[6,5,4,3,2,1]
[1,2,3,4,5,6]

如果你省略第一个索引，切片将从列表头开始。如果你省略第二个索引，切片将会到列表尾结束。 所以如果你两者都省略，切片就是整个列表的一个拷贝。由于列表是可变的，通常在修改列表之前，对列表进行拷贝是很有用的。

切片运算符放在赋值语句的左边时，可以一次更新多个元素：

t = ['a', 'b', 'c', 'd', 'e', 'f']
t[1:3] = ['x', 'y']
print(t)
['a', 'x', 'y', 'd', 'e', 'f']

列表和字符串

一个字符串是多个字符组成的序列，一个列表是多个值组成的序列。但是一个由字符组成的列表不同于字符串。可以使用 list 将一个字符串转换为字符的列表:

s = 'spam'
t = list(s)
print(t)

其结果为：

[‘s’, ‘p’, ‘a’, ‘m’]

由于 list 是内置函数的名称，你应避免将它用作变量名。我同样避免使用 l ，因为它看起来很像1。这就是为什么我用了 t 。

list函数将字符串分割成单独的字符。如果你想将一个字符串分割成一些单词，你可以使用 split 方法：

s = 'I like sing,jump,rap and play basketball'
t = s.split()
print(t)

其结果为：

[‘I’, ‘like’ ,’sing,jump,rap’ ,’and’,’play’,’basketball’]

可以提高一个叫做 分隔符（delimiter） 的可选参数，指定什么字符作为单词之间的分界线。下面的例子使用连字符作为分隔符：

s = 'spam-spam-spam'
delimiter = '-'
t = s.split(delimiter)
print(t)

其结果为：

[‘spam’, ‘spam’, ‘spam’]

join的功能和 split 相反。它将一个字符串列表的元素拼接起来。join 是一个字符串方法，所以你需要在一个分隔符上调用它，并传入一个列表作为参数：

t = ['pining', 'for', 'the', 'fjords']
delimiter = ' '
s = delimiter.join(t)
print(s)

其结果为：

‘pining for the fjords’

在这个例子中，分隔符是一个空格，所以 join 在单词之间添加一个空格。如果不使用空格拼接字符串，你可以使用空字符串 ‘’ 作为分隔符。

通过阅读文档，可以得到math标准库，math包中补充了更多的数学函数。在使用时只需要导入包即可。
示例为math包内的两个常数：

import math
print(math.pi)
print(math.e)

结果输出：

3.141592653589793
2.718281828459045

当然，如果想要更加高级的数学功能，可以考虑选择标准库之外的numpy和scipy项目，它们不但支持数组和矩阵运算，还有丰富的数学和物理方程可供使用。

函数 说明
math.acos(x) 反三角函数的反余弦函数，返回弧度为x的反余弦值
math.asin(x) 反三角函数的反正弦函数，返回弧度为x的反正弦值
math.atan(x) 反三角函数的反正切函数，返回弧度为x的反正弦值
math.ceil(x) 对传入的参数进行向上取整，例如x=2.7，返回3
math.cos(x) 三角函数的余弦函数，返回弧度为x的余弦值
math.degrees(x) 把弧度x转化为角度，例如x=math.pi，返回45.0
math.fabs(x) 对传入的参数进行绝对值，例如x=-4，返回4
math.factorial(x) 对传入的参数进行阶乘，例如x=4，返回24
math.floor(x) 对传入的参数进行向上取整，例如x=2.7，返回2
math.gcd(x,y) 返回x,y的最大公约数，例如x=3,y=15，返回3
math.hypot(x,y) 返回(x+y)^2的值，效果和勾股定律一致，例如x=3，y=4，返回5.0
math.log(x) 对数，默认基底为e。可以使用base参数，来改变对数的基底。比如math.log(100,base=10)
math.modf(x) 返回由x的小数部分和整数部分组成的元组,例如x=math.pi，返回(0.14159265358979312, 3.0)
math.pow(x,y) 指数运算，得到x的y次方，与x**y效果类似
math.radians(x) 把角度x转化为弧度，例如x=45，π/4
math.sin(x) 三角函数的正弦函数，返回弧度为x的正弦值
math.sqrt(x) 对传入的参数进行平方根，例如x=4，返回2
math.tan(x) 三角函数的正切函数，返回弧度为x的正切值
math.trunc(x) 返回x的整数部分,效果与x//1相似

random包

random包是可以用来生成随机数。随机数不仅可以用于数学用途，还经常被嵌入到算法中，用以提高算法效率，并提高程序的安全性。

函数 说明
random.choice(x) 在序列x中随机选取一个值，例如random.choice(range(20)),从0到19中随机挑选一个整数
random.randrange(x,y,z) 从x到y-1间随机选取z个值，y值为必需，x、z可选择
random.random() 在[1,0)间随机挑选1个值
random.randint(x) 在[0,x)中随机挑选1个整数
random.seed(x) 可选参数x，无设置时以时间戳为种子
random.shuffle(list) 打乱列表list内原有排序
random.uniform(x, y) 在[x,y]间随机挑选1个值的下一个实数

Python的编码

Python的默认编码是UTF-8,所有字符串都是 unicode 字符串，如果需要在文件应用不同类型的编码，需要在文件头添加

# -*- coding:UTF-8 -*-

描述使用的是GBK编码。

Python的标识符

标识符可以理解为变量的名字，可以用中文作为标识符，最好使用英语作为标识符，避免不同操作系统导致的乱码。使用英语作为标识符时最好是见文之意，方便后续的维护。最好遵循以下的命名规制：

必须以字母表中的字母或者下划线（_）开头
其他部分可由字母、数字、下划线组成
大小写敏感，例如result、RESULT是表示不同的标识符

Python的保留字

Python中具有32个保留字，这些保留字不可以作为标识符，否则会引起异常。保留字如下表：

False None True and as assert break class
continue def del elif else except finally yield
for from global if import in is lambda
no local not or pass raise return try

Python的第一个程序

每种编程语言都是从hello world开始编程之旅，这次我也不例外。

# -*- coding:UTF-8 -*-
print('hello world')    #第一个程序

结果为：

hello world

由此可见，在文件可以使用#来进行注释，其注释结果不被进行编译。多行注释可以用多个#号，还有’’’和”””，注意当使用三单（双）引号时必需有配对才可以正常使用，例如

"""hello world,  
bye,world."""

print()函数则是将括号内的内容进行打印，如果是使用编译器，则是在编译器的控制台出打印内容，若是在终端进行命令行操作，则在命令行的下一行进行打印内容。

程序的结构

程序的结构有顺序结构、循环结构、选择结构。
循环结构和选择结构中还涉及行与缩进，缩进的空格数是可变的，但是同一个代码块的语句必须包含相同的缩进空格数。
如果在连续的代码行中的缩进空格数相等，那么这些代码行属于同一个代码块。这一点在选择结构、循环结构、函数、类中得以应用。函数和类将在后续内容中进行讲解。

顺序结构

顺序结构是一种线性、有序的结构,它依次执行各语句模块。

a=500
b=20
print(a+b)

结果为：

520

选择结构

选择结构是根据条件成立与否选择程序执行的通路。
选择结构的内容可以理解为如果……就……或者……否则……，用程序表示则为if……else if……else……，用程序表示：

a=520
b=250
if a==520:
    print(’我爱你‘)
elif a==250:
    print('你个大傻子')
else:
    print('没有结果')

结果为：

我爱你

当然elif、else语句不是一定要有的，但是如果有elif或else语句，那一一定要有if语句。

值得一提的是，在选择结构中存在断点的的想象，符合逻辑的推导。例如：

a=520
b=250
if a==520:
    print(’我爱你‘)
elif b==250:
    print('你个大傻子')
else:
    print('没有结果')

结果为：

我爱你

在结构中，a和b同时符合选择分支，但是只能选择一条通路，这时就得判断先后顺序来选择通路。

循环结构

循环结构是重复执行一个或几个模块,直到满足某一条件为止。
break语句可以使循环结构终止，如果有多层循环的话，只能使最内层循环终止。
continue语句可以使循环结构跳过本次循环，一般和条件结构一起使用。
pass语句可以使循环结构进入循环阻塞，按Ctrl+C进行终止。

while循环

while 判断条件(condition)：
执行语句(statements)……
当判断条件不符合时，while循环结束，结果停止在最后一次循环上。
例如：

i=1
while i<=60:
    print("我向你发射"+str(i)+"秒的爱意！biu～")
    i=i+1

当i==61时，循环停止，其结果为：

我向你发射1秒的爱意！biu～
我向你发射2秒的爱意！biu～
……
我向你发射60秒的爱意！biu～

因为i的类型为int，使用str()可以将i转化为str。

break语句示例：

i=1
while i<=60:
    print("我向你发射"+str(i)+"秒的爱意！biu～")
    break；

其结果为：

我向你发射1秒的爱意！biu～

continue语句示例：

i=1
while i<=60:
    print("我向你发射"+str(i)+"秒的爱意！biu～")
    if i!=60:
        continue;

其结果为：

我向你发射1秒的爱意！biu～
我向你发射60秒的爱意！biu～

pass语句示例：

i=1
while i<=60:
    pass    
    print("我向你发射"+str(i)+"秒的爱意！biu～")

其结果为：

我向你发射1秒的爱意！biu～
我向你发射1秒的爱意！biu～
……
我向你发射1秒的爱意！biu～

仅有按下Ctrl+C后才终止，否则无限循环，直到CPU空间被占满。
while死循环示例：

while i==1:     # or （while Ture：）

    print("我向你发射爱意！biu～")

其结果为：

我向你发射爱意！biu～

for循环

for循环可以遍历任何序列的项目，如一个列表或者一个字符串。

for <variable\> in <sequence\>:
    <statements\>
else:
    <statements\>

例如：

for i in range(5):
    print('一如既往地想你+'+str(i))

结果为：

一如既往地想你+0
一如既往地想你+1
一如既往地想你+2
一如既往地想你+3
一如既往地想你+4

range()函数可以使括号内的参数生成一个列表，具有多种使用方法。

  • range(5),将产生含0~4的数值列表[0,1,2,3,4]
  • range(2,5),将产生含2~4的数值列表[2,3,4]
  • range(2,5,2),将产生含2~4且步长为2的数值列表[2,4]
