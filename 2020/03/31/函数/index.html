函数

当我们在进行编程的时候，如果只是按照顺序结构一步一步地写下去，像List那篇文章的小样例一样，那么当存在内容重复的代码块的时候，是不是内容杂冗呢。如果可以把重复的代码块进行命名，当调用的时候，直接调用岂不是简洁明了呢。这时候函数就屁颠屁颠地跑过来说：“你终于想起我了”。

定义

函数是可重复使用的、用来实现单一或相关联功能的代码段。

函数能提高应用的模块性、提高程序可读性和代码的重复利用率。Python提供了许多内建函数，比如print()。但你也可以自己创建函数，这被叫做用户自定义函数。

规则

在定义函数的时候，必须遵循Python定义函数的规则，不然会被视为错误：

  • 函数代码块以def关键词开头，后接函数标识符名称（即函数名）和圆括号 ()。
  • 任何传入参数和自变量必须放在圆括号中间，圆括号之间可以用于定义参数。
  • 函数的第一行语句可以选择性地使用文档字符串—用于存放函数说明。
  • 函数内容以冒号起始，并且缩进。
  • return [表达式] 结束函数，选择性地返回一个值给调用方。不带表达式的return相当于返回 None。

语法

def 函数名（参数列表）:
    函数体

参数传递

在定义函数的时候，我们把参数的名字和位置确定了下来，函数的定义就完成了。对于函数的调用者来说，只需要知道如何传递正确的参数，以及函数将返回什么样的值就够了，函数内部的复杂逻辑被封装起来，调用者无需了解。

Python的函数定义非常简单，但灵活度是非常大的。除了正常定义的必选参数外，还可以使用默认参数、可变参数和关键字参数，使得函数定义出来的接口，不但能处理复杂的参数，还可以简化调用者的代码。

单一位置参数

def power(x):
    return x * x

对于power(x)函数，参数x就是一个位置参数。

当我们调用power函数时，必须传入有且仅有的一个参数x即可：

>>> power(2)
4
>>> power(10)
100

多位置参数

def power(x, n=2):
s = 1
while n > 0:
    n = n - 1
    s = s * x
return s

对于这个求x的n次幂函数，其间涉及到了默认参数，当仅传入一个参数时，是返回x的平方，当传入两个参数时返回x的n次幂。

>>> power(2)
4
>>> power(2,3)
8

不定长参数

你可能需要一个函数能处理比当初声明时更多的参数。这些参数叫做不定长参数。

def printinfo_1( a, *b ):
    "打印任何传入的参数"
    print ("输出: ")
    print (a)
    print (b)
def printinfo_2( a, **b ):
    "打印任何传入的参数"
    print ("输出: ")
    print (a)
    print (b)

其结果为：

>>>printinfo_1(1)
>>>1
>>>printinfo_1(1,2,3)
>>>1
>>>(2,3)
>>>printinfo_2(1)
>>>1
>>>printinfo_2(1,a=2,b=3)
>>>1
>>>{'a':2,'b':3}

如果细心的话，会发现，由于第二个参数的*的个数导致返回值类型的不同。单是单个*时，除第一个参数正常返回外，多余参数使用元组进行返回；一对*时，在传入参数是加以变量传入，则返回字典形式。

匿名函数（lambda）

通过lambda关键字进行链接表达式，使其成为匿名函数，不用再使用def关键字，该函数拥有自己的命名空间，且不能访问自己参数列表之外或全局命名空间里的参数。

sum = lambda a,b :a+b
print(sum(1,2))

其结果为：

3

return语句

一些函数具有返回数据的作用，return [表达式] 语句用于退出函数，选择性地向调用方返回一个表达式。不带参数值的return语句返回None。

def function(a):
    result = a+1;
    print("函数内"，result)
result = function(10)+1
printf("函数外"，result)

其结果为：

函数内 :  11
函数外 :  12

小样例

#!/usr/bin/python      
# -*- coding: utf-8 -*-     
'''打印文件列表'''
import os
def print_filelist(path):
    """传入一个文件路径，打印该路径下的所有文件和文件夹"""
    try:
        files = os.listdir(path)
        for file in files:
            print(file)
    except:
        print("输入路径不存在")
if __name__=="__main__":
    path = input("请输入文件路径:")
    print_filelist(path)
